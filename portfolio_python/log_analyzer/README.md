# Log Analyzer (Python CLI)

A command-line based log analysis tool built using Python.  
The application reads server log files and provides a summary of log levels
such as INFO, WARNING, and ERROR.

## Features
- Counts total log entries
- Categorizes logs by severity
- Identifies repeated error messages
- Handles missing file errors gracefully

## Technologies Used
- Python
- File handling
- Dictionaries and string processing

## How to Run
1. Clone the repository
2. Ensure a log file exists (sample.log provided)
3. Run `python log_analyzer.py`
