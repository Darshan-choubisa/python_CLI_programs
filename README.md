# Python_CLI_Programs

This repository contains a collection of Python command-line interface (CLI) projects built to strengthen core Python concepts, problem-solving skills, and real-world logic implementation.  
Each project focuses on practical use cases such as data management, file handling, and text processing.

### 1. Contact Book (Python CLI)
A command-line based contact management application that allows users to add, view, update, and delete contacts.  
The project uses file handling to store contact data persistently and demonstrates structured program flow and input validation.

**Key Concepts:**
- CRUD operations
- File handling
- Functions and lists
- User input validation

---

### 2. Log Analyzer (Python CLI)
A Python-based log analysis tool designed to read server log files and generate meaningful summaries.  
It categorizes logs by severity levels such as INFO, WARNING, and ERROR, and identifies repeated error messages.

**Key Concepts:**
- Text file parsing
- Dictionary-based data aggregation
- String manipulation
- Error handling

---

### 3. Expense Tracker (Python CLI)
A command-line expense tracking application that helps users record daily expenses and analyze spending patterns.  
The project supports monthly and category-wise summaries using CSV-based persistent storage.

**Key Concepts:**
- CSV file handling
- Data summarization
- Modular functions
- Real-world data processing
